# מדריך הפעלה — מכונת כביסה דגם לדוגמה WM-2000

> קובץ דוגמה. החליפי במדריכים האמיתיים שלך (טקסט או Markdown) באותה תיקייה.

## הפעלת תוכנית כביסה
1. ודאו שדלת התוף סגורה היטב עד נקישה.
2. פתחו את ברז המים שמאחורי המכונה.
3. הוסיפו אבקה או נוזל כביסה למגירת החומרים: התא השמאלי לכביסה ראשית, התא הימני למרכך.
4. סובבו את חוגת התוכניות לתוכנית הרצויה (כותנה 40°, סינטטי, עדין, מהיר 30 דקות).
5. לחצו על כפתור ההפעלה. נורית ההפעלה תידלק והמכונה תתחיל למלא מים.

## טמפרטורה וסחיטה
- כוונון הטמפרטורה נעשה בכפתור הטמפרטורה משמאל לחוגה. טווח: קר עד 90°.
- מהירות הסחיטה נקבעת בכפתור הסחיטה. לבדים עדינים מומלץ עד 800 סל"ד.

## קודי תקלה נפוצים
- **E1** — בעיית ניקוז. בדקו שצינור הניקוז אינו חסום או מכופף, ונקו את מסנן המשאבה.
- **E2** — הדלת אינה נעולה. סגרו שוב את הדלת היטב עד נקישה.
- **E3** — אין כניסת מים. ודאו שברז המים פתוח ושמסנן הכניסה אינו סתום.

## ניקוי מסנן המשאבה
1. כבו את המכונה ונתקו מהחשמל.
2. פתחו את הדלתית הקטנה בתחתית המכונה מצד ימין.
3. הניחו מטלית לספיגת מים, ושחררו את פקק המסנן נגד כיוון השעון.
4. הוציאו לכלוך ושיער, שטפו את המסנן במים זורמים, והחזירו אותו חזרה.

## בטיחות
- אין לפתוח את הדלת בזמן שהתוף מסתובב או כשיש מים בתוף.
- כל תקלה חשמלית או דליפת מים מתמשכת — נתקו מהחשמל ופנו לשירות היצרן. אין לפרק רכיבים חשמליים עצמאית.
